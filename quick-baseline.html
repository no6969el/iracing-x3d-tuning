Quick Baseline — just the fixes

Install one free app, click four things, run one file, reboot. This is the whole baseline — no reading required. Graphics polish comes later, once you're smooth.

⬇ Download the kit (.zip) Full guide (the “why”) Guided walkthrough
⚠️ The one-shot optimizer changes Windows settings (registry, power, Defender). Everything is reversible (undo scripts in the kit), and nothing runs without an admin prompt you approve. Unzip the kit to a permanent folder first — e.g. C:\iRacingTuning\.

1Install Process Lasso (free)

Get it from bitsum.com and install. The free version does everything we need — no trial, no purchase.

2Set it up — the only clicking you'll do

  1. Power plan: Process Lasso Main menu → Power → activate “Bitsum Highest Performance.” (Keeps all cores unparked — what a dual-CCD chip needs.)
  2. Launch iRacing and enter a race (any test or AI session works) so iRacingSim64DX11.exe appears in Process Lasso's process list. You can exit the session after — the process just has to have been seen once.
  3. Pin the sim: right-click iRacingSim64DX11.exe → CPU Sets (⚠️ not CPU Affinity) → select your V-Cache cores:
    Your CPUSelect cores
    9950X3D / 7950X3D0–15
    9900X3D / 7900X3D0–11
  4. Protect it: right-click iRacingSim64DX11.exe → ProBalance → exclude it (so it's never throttled).

3Run the one-shot optimizer

Double-click Apply-Baseline.bat in the kit folder. It confirms your CPU (16- or 12-core), shows a short advisory, then applies all five script fixes in one window with one admin prompt — no typing:

  • Defender exclusions (stops mid-race file scans)
  • USB Selective Suspend + Game Bar off
  • Diagnostic log on (for the stutter scanner)
  • Timer fix — the VR “stutters until I click” problem
  • GPU interrupts off the sim's core

Then reboot. The GPU-interrupt and timer fixes only take effect after a restart.

4Prove it worked (optional, 10 min)

After the reboot, run Apply-Baseline.bat again and press T to launch FullTrace (or run scripts\FullTrace.ps1 directly). Race ~10 minutes, press Ctrl+C, and open the CSV it saves to your Desktop: no skipped seconds in the timestamp column = no stalls.

Later

Graphics last — once you're smooth

When the baseline feels good, do the visual pass: iRacing's ini tweaks and the NVIDIA Control Panel values (full guide, fixes #9–#10), and the VR clarity section if you race in a headset. Doing graphics after the baseline means any new stutter has an obvious cause.

Want each step explained, or something didn't work? The guided walkthrough holds your hand, and Start-Tuning-Menu.bat includes troubleshooting tools and undo options for everything.
