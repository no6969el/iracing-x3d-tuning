# iRacing Tuning — Ryzen X3D + NVIDIA

Zero hiccups in iRacing on Ryzen X3D — dual-CCD (9950X3D / 7950X3D / 9900X3D / 7900X3D) **and** single-CCD (7800X3D / 9800X3D / 5800X3D) 

5600X3D NOT SUPPORTED YET.

Six steps, measured on a real rig, all reversible. The guide and menus detect your chip and only ever show the fixes that are safe for it.
